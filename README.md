# Internship-Management-System
A web-based application designed to manage student internships efficiently. It allows administrators, tutors, and students to organize and monitor internship activities, including company management, evaluations, incidents, and daily reports. Built using Java Servlets, JSP, and MySQL following the MVC architecture.

## Requirements
- **Eclipse IDE:** Eclipse IDE for Enterprise Java and Web Developers (version 2023-09 or later)  
- **Java Development Kit (JDK):** Version 17 or higher (fully tested with JDK 21)  
- **Apache Tomcat:** Version 9.0 or higher as the runtime server  
- **MySQL Server:** Version 8.0 or higher for the database backend  
- **JDBC Connector:** MySQL Connector/J (compatible with your MySQL version)
